## Apache Camel
- Software integratie framework
- Talloze componenten
- Geprogrammeerd in Java!
- Verschillende DSL's, waaronder XML en Java
- Beschikbaar als Maven artifacts
- Standalone of integreren met Spring

![camel-logo](./pics/apache-camel-logo.png)
